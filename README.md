# Список дел (To-Do List)

Простая консольная программа для управления задачами. Запрашивает количество дел, затем сами дела и выводит их нумерованным списком.

## Возможности
- Добавление задач
- Вывод списка с нумерацией
- Защита от некорректного ввода

## Технологии
- Python 3
- Работа со списками
- Обработка ошибок

## Как запустить
1. Скачай файл `todo_list.py`
2. Открой терминал в папке с файлом
3. Выполни: `python todo_list.py`

## Планы по улучшению
- Сохранение задач в файл
- Возможность удалять выполненные задачи
